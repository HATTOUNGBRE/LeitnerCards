## Backend

API NestJS de LeitnerCards.

Ce premier push se concentre sur `cards` pour la gestion des fiches.

Les modules `authentication` et `users` sont conserves car ils appartiennent a une autre partie du projet.

## Installation

```bash
npm install
```

## Demarrage

```bash
npm run start
npm run start:dev
npm run start:prod
```

## Tests

```bash
npm run test
npm run test:e2e
npm run test:cov
```
